# Locally Weighted Regression (LWR)

## Features
- Non-linear regression using local weighting
- Gaussian kernel-based weighting
- Smooth curve fitting

## Concept
Locally Weighted Regression (LWR) is a non-parametric regression method that fits a model around a query point using nearby data points.

## Dataset
Synthetic dataset (sin function with noise)

## How to Run
pip install numpy matplotlib
python lwr.py
